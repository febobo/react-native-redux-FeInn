### Requirements
* node  
* xcode  

### Usage
```
$ git clone https://github.com/febobo/react-native-redux-FeInn.git
$ cd react-native-redux-FeInn
$ npm install
```

### For Ios 
```
$ npm start
``` 
use xcode open react-native-redux-es6/ios/demo.xcodeproj, then click run  
or ```$ open FeInn.xcodeproj```,then click run
